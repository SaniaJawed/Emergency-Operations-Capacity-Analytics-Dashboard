# 🚑 Emergency Operations & Capacity Analytics Dashboard

An interactive healthcare analytics dashboard designed to monitor Emergency Department (ED) performance, patient flow, hospital capacity, and clinical outcomes. This project combines Power BI analytics with a responsive HTML, CSS, and JavaScript dashboard to deliver an engaging, data-driven storytelling experience.

## 📌 Project Overview

Emergency Departments operate in highly dynamic environments where patient demand, resource availability, and treatment efficiency directly impact healthcare outcomes.

This project provides an interactive analytics dashboard that enables users to explore emergency department performance through operational KPIs, patient flow metrics, hospital benchmarking, demand patterns, and clinical outcome analysis.

The dashboard transforms healthcare data into actionable insights using interactive visualizations and business storytelling.

---

## 🎯 Objectives

- Monitor Emergency Department performance
- Analyze patient flow across the care journey
- Evaluate hospital operational efficiency
- Identify bottlenecks affecting treatment time
- Compare hospital performance using key metrics
- Support data-driven operational decision making

## 📌 Key Performance Indicators

- Total Patient Visits
- Average Waiting Time
- Bed Occupancy Rate
- Readmission Rate
- Mortality Rate
- Patient Satisfaction
- Average Length of Stay
- Treatment Delay

---

## 🔍 Key Insights

- Emergency admissions account for the largest share of patient visits.
- Seasonal demand impacts waiting times and operational pressure.
- Readmission rates vary significantly across hospitals.
- Patient satisfaction is strongly associated with operational efficiency.
- Treatment delays represent one of the primary operational bottlenecks.
- Hospital performance varies across multiple clinical and operational indicators.
